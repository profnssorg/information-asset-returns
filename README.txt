Este programa faz a análise do efeito de noticias politicas no cupom cambial


Há dois arquivos main, um para coletar as noticias e outro para fazer todo o resto da analise. Rodar primeiro o das noticias, e esperar o processo de coleta terminar. O processo pode ser demorado. Após o processo de scrapping terminar, rodar o outro main, que fará todo o resto da coleta, processamento e saída de dados.
